# Bizuno Payments for WooCommerce

Development repository for the WordPress plugin **Bizuno Payments for WooCommerce** (WordPress.org slug: `bizuno-payments-for-woocommerce`).

Adds two WooCommerce payment methods:

- **PayFabric credit-card gateway** (Global Payments) — hosted payment page, live + sandbox modes.
- **Purchase Order** — offline business PO method.

Works standalone in any WooCommerce store, and integrates with the Bizuno Accounting/ERP platform for back-office reconciliation.

- **Requires:** WordPress 6.5+, PHP 8.0+, WooCommerce 8.0+
- **HPOS:** compatible (custom order tables)
- **License:** AGPL-3.0-or-later
- **External service:** processes credit cards via PayFabric (https://www.payfabric.com). See the "External services" section of [`readme.txt`](readme.txt).
- **WordPress.org listing:** the canonical, directory-facing readme is [`readme.txt`](readme.txt).
